# 2B-Web-Scraping
This project uses BeautifulSoup and Selenium to scrape 2B's website (Electronic devices shop). 

The python script scrapes all the data in an excel sheet:
    - Categories
    - Subcategories
    - Products: - Brand
                - Name
                - URL
                - Description
                - Price
                - Rating
                - Stock
                - Reviews
                - Image
